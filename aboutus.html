Gianatics Developers Limited is a quad group consisting of students from BSIT-WMA TW22. The primary objective of the group is to redesign the current website of the Pokemon Go Web Store. The group logo consists of a piece from a chess board which symbolizes trickiness since the project is tricky. The initials "G" and "N" are present to symbolize the word "Gianatics", the group name. The curly brackets symbolize the proper, majestic design of the website by implementing beautiful css design. The "<>" symbolizes the HTML entities which the group used all throughout the code. Lastly the logo is surrounded outside by the group name, the year of development (in binary code to make it look cool), and lastly the name of the university the group is enrolled.
